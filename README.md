# 🛒 Ecommerce Dashboard (React + Vite)

A modern **Ecommerce Dashboard** built using **React, Vite, Tailwind CSS and FakeStore API**.
This project demonstrates a complete **frontend ecommerce workflow** including authentication, product listing, cart management, and user profile handling.

---

# 🚀 Live Demo

Deployed on Vercel
🔗 https://ecommerce-dashboard-eight-teal.vercel.app/

---

# 📌 Features

### 🔐 Authentication

* User Registration
* User Login
* Email already registered validation
* Password validation
* Redirect after successful registration
* Protected routes

### 🛍 Products

* Fetch products from FakeStore API
* Product cards with image, title and price
* Add products to cart
* Loading spinner while fetching API

### 🛒 Cart System

* Add to cart
* Prevent duplicate items
* Increase / decrease quantity
* Remove item from cart
* Item subtotal
* Cart total calculation
* Cart badge with live count

### 👤 User Profile

* Display user details
* Edit user profile
* Update name, email and password
* Logout functionality

### 🎨 UI / UX

* Tailwind CSS modern UI
* Responsive design
* Shared Navbar across pages

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/yourusername/ecommerce-dashboard.git
```

Navigate into the project folder

```
cd ecommerce-dashboard
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```
